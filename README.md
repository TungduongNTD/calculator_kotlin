# Calculator Kotlin

This is a simple yet functional calculator application developed using Kotlin for Android. The app is designed to handle basic arithmetic operations with a user-friendly interface, allowing for efficient and accurate calculations.

## Features
- **Basic Arithmetic Operations:** Perform addition (+), subtraction (-), multiplication (ร), and division (รท).
- **Multi-step Calculations:** Supports calculations with up to three operands.
- **Decimal Support:** Handles both integer and decimal inputs with precision.
- **Clear and Backspace Functions:** Easily correct mistakes or reset the calculation.
- **Dot Input:** Allows decimal input for more accurate results.
- **Live Calculation Display:** Displays the calculation steps in real-time in a secondary view.

## Technologies Used
- **Kotlin:** The primary language for developing Android applications.
- **Android Studio:** The development environment used for the project.
- **ViewBinding:** Used to simplify UI interaction in the app.

## How to Run the Project
1. Clone this repository to your local machine:
    ```bash
    git clone https://github.com/TungduongNTD/calculator_kotlin.git
    ```
2. Open the project in **Android Studio**.
3. Build and run the application on an Android emulator or a physical device.
